1.0.0.Final Released July 15, 2013

This was the 1.0 public release of runtime governance which included the following features:

  • Embedded activity collector for SwitchYard
  • REST based Activity Server and Store (JDBC)
  • Event Processing, using Java, Drools rules, MVEL
  • Active Collections for presenting results
  • Gadget Server based UI
